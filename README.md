## 📋 What is mnemosyne?

mnemosyne is a tool to give AI agents a memory. It helps multiple AI agents learn, share knowledge, and build on past experiences. This makes the AI system better at completing tasks, understanding information, and working together.

The app uses a system called a knowledge graph to organize memories. It also connects these memories using special codes called embeddings. These codes help the AI find related information quickly.
